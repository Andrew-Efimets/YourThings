Веб-приложение для публикации и поиска объявлений о продаже товаров и услуг. Реализовано с поддержкой геолокации, фильтрации и сортировки товаров.

Основной функционал

·       Авторизация: регистрация, вход, восстановление пароля.

·       Объявления: создание, редактирование, удаление.

·       Поиск и фильтры: по категориям, городам, цене, названию.

·       Личный кабинет: управление своими товарами, покупками и профилем, оплата на сайте (Stripe).

·       Уведомления: получение e-mail писем для восстановления пароля, при создании объявления, при продаже товара.

Технологический стек

·       Frontend: HTML5, CSS3, JavaScript (Blade).

·       Backend: PHP (Laravel).

·       База данных: MySQL, Redis.

·       Дополнительно: Docker, Git, API Yandex, API Stripe, MailDev.
